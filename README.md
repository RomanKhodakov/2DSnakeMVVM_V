# 2DSnakeMVVM
Мой пятый проект по изучению Unity для тестированния архитектуры Model-View-ViewModel (MVVM).

Создан на как мини-копия четвертого проекта, с тем отличием, что вместо MVC использовался MVVVM.

Вокруг игрока создаются бонусы, подбирая которые его счёт увеличивается. Счёт выводится через UI панельку.

Бонусы и игрок создаются через Factory Method. Созданные бонусы хранятся в Object pool'е.
